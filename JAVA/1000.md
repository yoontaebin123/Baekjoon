# A+B
+ 두 정수 A와 B를 입력받은 다음,A+B를 출력하는 프로그램을 작성하시오.
```java
import java.util.Scanner;

public class Baekjoon {

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int num1;
		int num2;
		num1 = sc.nextInt();
		num2 = sc.nextInt();
		System.out.println(num1 + num2);

	}

} 
```
## 결과
***
#### 입력
```
1 2
```
#### 출력
```
3
```
