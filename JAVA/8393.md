# 합
+ n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.
```java
import java.util.Scanner;

public class Main {

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int n = sc.nextInt();
		int sum = 0;
		for(int i = 1; i <= n; i++) {
			sum += i;
		}
		System.out.println(sum);
		
		

	}

}
```
## 결과
#### 입력
```
3
```
#### 출력
```
6
```
